
# Employee Management System (CLI)

A command-line employee management system built using Node.js.

## Features

- Add Employee
- View Employees
- Update Employee
- Delete Employee
- Search Employee
- File persistence using JSON

## Requirements

- Node.js 18+

## How to Run

1. Clone the repository:

git clone https://github.com/Priyanshu9817/Experiment-4.A.git

2. Navigate into folder:

cd app.js

3. Run the program:

node app.js
